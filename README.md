# SmartOps Infrastructure (Terraform)

This repository contains Infrastructure as Code (IaC) using Terraform.

## Responsibilities
- AWS VPC and networking
- Public and private subnets
- EC2 instances
- Security groups
- S3 (state storage, artifacts)
- DNS setup (future)

## Goal
Provision all infrastructure automatically using Terraform.

## Structure (Planned)
- modules/
- environments/
- main.tf
- variables.tf
- outputs.tf

## Notes
All infrastructure must be reproducible and version-controlled.
